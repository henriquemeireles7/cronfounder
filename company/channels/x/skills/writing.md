# Writing for X

Read by content builders before drafting for this surface.

- First line earns the second line; no throat-clearing.
- One idea per post. Threads: each post must stand alone when screenshotted.
- Concrete numbers beat adjectives. "40 → 214 stars in 12 days" beats "growing fast".
- No hashtags, no engagement bait, no "🧵👇".
- Voice comes from doctrine/identity.md — this file only covers the surface's format rules.
- Links go in the final post of a thread, never the first.
